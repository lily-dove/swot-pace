Welcome to the code for the manuscript "Biophysical Dynamics of Mesoscale Eddies: Coincident Observations from SWOT and PACE" by Lilian A. Dove and Mara A. Freilich!

Here, you will find the code used to generate figures in the manuscript as well as to do your own initial data investigations with SWOT and PACE data.

Initial example data is provided in this repo in the PACE, SWOT, and DUACS folders. Instructions to download data are provided at the start of the Jupyter notebook.
